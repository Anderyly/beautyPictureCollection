# beautyPictureCollection

批量采集美女写真等图片到typecho上，图片可选择本地化

👉 http://52rm.cc

# 编译
```shell
    git clone https://github.com/Anderyly/beautyPictureCollection.git
    go mod init && go mod tidy
    go build
```

# 运行
### 请配置set.ini内容
```shell
    chomd +x warm
    ./warm
```
